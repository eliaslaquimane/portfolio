Elias Alberto Laquimane

I am a web developer with over 4 years of experience creating intuitive interfaces. I hold a Bachelor's degree in Chemistry Education from Rovuma University in Nampula, Mozambique. My primary focus is Front-end development using HTML, CSS, JavaScript, React Native, and TypeScript. I also work as a UI Designer when needed, utilizing tools like Figma, Adobe XD, and Lunacy to create user-centered solutions. I am continuously seeking to expand my knowledge and deliver end-to-end value. Key Interests: Front-end Development (React, Flutter, Python, TypeScript), UI/UX Design (Figma, Adobe XD, Lunacy). Objective: To create digital products that positively impact people's lives, continuously enhance my skills, and learn new technologies.

Works and projects

Biblioteca Unirovuma

Biblioteca Unirovuma is a web application designed to library operations, including book management, user registration, and borrowing processes. It provides an intuitive interface for both librarians and users, enhancing the overall library experience.

HTML CSS Bootstrap

NutryUnir

NutryUnir is a mobile application aimed at promoting healthy eating habits among university students. It provides personalized meal plans, nutritional information, and a platform for users to share their recipes and experiences.

React Native TypeScript

PEA-Quimica

PEA-Quimica is a mobile application designed to assist students in learning chemistry concepts through interactive simulations and quizzes. It aims to make chemistry more accessible and engaging for university students.

HTML CSS Bootstrap

Hotel-link

Hotel-link is a web application designed to connect travelers with hotels and accommodations. It provides a user-friendly interface for searching, booking, and managing hotel reservations, making travel planning more convenient and efficient.
